CONTAINER AND FLEXBOX

FlexBox

Flexbox — containerlarni joylashtirishda ishlatiladi, yordamida elementlarni turli xil joylashtirish mumkin.

Container

CSS containerlari bu shrifti, rangi, fon rangi, width, height kabi xususiyatlarga ega bo'lgan tarkibni joylashtirish uchun ishlatiladi.

space-evenly

space-evenly — barcha elementlar orasidagi bo‘shliqni teng taqsimlaydi.

space-between

Elementlarni chap va o‘ngdan joylashtirib, qolgan bo‘shliqni teng taqsimlaydi.

align-items

Elementlarni vertikal bo‘yicha joylashtirish imkonini beradi: yuqori, o‘rta yoki pastga.

FlexBox

Flexbox yordamida elementlarni kerakli tartibda joylashtirish mumkin.
